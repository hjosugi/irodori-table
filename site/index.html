Why Irodori Table

日常の DB 作業を速く、長く使える形で。

TablePlus の軽さ、DataGrip 級の編集力、DBeaver 級の接続範囲を、オープンな Rust/Tauri アプリとして積み上げるプロジェクトです。

軽量なデスクトップ

Rust コアと Tauri シェルで、起動、接続、結果表示の待ち時間を抑えます。

広いデータソース

SQL だけでなく、document、graph、time-series、lakehouse まで拡張できる設計です。

キーボード中心

リマップ可能なショートカット、Vim モード、コマンドパレットを初期から重視します。

大きな結果に強い

ストリーミング、上限付き取得、仮想化グリッドで巨大テーブルでも軽く扱います。

接続と安全性

直接接続、SSH、SOCKS/HTTP、複数段プロキシ、OS キーチェーンを見据えます。

拡張できる基盤

拡張 manifest、typed API、ドライバ、テーマ、結果ビューを段階的に公開します。

Data sources

まず主要 DB、次にすべてのデータ作業へ。

最初の製品価値は SQL GUI としての速さです。内部の adapter contract は、 ドキュメントストアやグラフ DB も同じ作業台に載せられるように設計します。

Roadmap

縦切りから daily-driver へ。

Phase 1

Thin vertical slice

SQLite / PostgreSQL 接続、SQL 実行、ストリーミング結果、オブジェクトブラウザ。

Phase 2

Power-user editor

Vim、キーバインド、複数ペイン、保存セッション、クエリ履歴、i18n。

Phase 3

Completion and intelligence

スキーマ認識補完、hover inspection、dialect-aware ranking、任意の AI provider。

Phase 4+

Coverage, network, extensions

Oracle、warehouse、graph、time-series、SSH/proxy、拡張 SDK、ローカル data API。
