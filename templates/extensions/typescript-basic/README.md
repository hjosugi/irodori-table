# TypeScript Basic Extension

This template is dual-licensed as `MIT OR 0BSD`.

It demonstrates:

- command registration;
- keybinding and result-grid action contributions;
- result-grid read + clipboard write through the host API;
- theme and SQL dialect registration;
- local dev fixtures and logs.

Run the local dev host from the repository root:

```sh
node packages/extension-sdk/bin/irodori-extension-dev.mjs templates/extensions/typescript-basic --once
```
