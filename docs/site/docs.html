Overview

現在の位置づけ

Irodori Table は開発中のオープンソース DB ワークベンチです。 Rust、Tauri、React、CodeMirror 6 で構成しています。

公開ステータス

現在は development preview です。安定版ではありません。

Quick start

ローカルで起動する

デスクトップアプリは apps/desktop にあります。 

cd apps/desktop
npm install
npm run dev

型生成、TypeScript、Vite build まで確認します。

cd apps/desktop
npm run build

公式サイトだけ確認します。

python3 -m http.server 8080 --directory docs/site

Connections

接続と対応データソース

SQL、document、key-value、warehouse 系を同じ接続層で扱います。 詳細な検証状況は data-source support status と implementation progress を参照してください。

PostgreSQL MySQL MariaDB SQL Server Oracle SQLite DuckDB MongoDB ClickHouse Snowflake BigQuery Redis Cassandra Bigtable InfluxDB

Query editor

クエリを書く、実行する

CodeMirror 6 ベースです。SQL dialect、Vim mode、format、comment toggle、 Run Current / selection 実行、接続別履歴を扱います。 

select c.id, c.name, sum(o.total) as lifetime_value
from customers c
join orders o on o.customer_id = c.id
where o.created_at >= now() - interval '90 days'
group by c.id, c.name;

Completion

軽量な schema-aware completion

補完はローカル metadata と現在文だけを浅く見ます。 schema、table、column、routine、keyword、alias、FK join を候補にします。

現在できること

schema.、alias.、JOIN 位置の候補を出します。 table / view / column hover は DDL、keys、indexes、comments、 quick sample を表示し、F12 または Ctrl/Cmd click で metadata へ移動できます。

Results

結果グリッドと export

結果はストリーミング、上限付き取得、row virtualization を前提にします。 export は CSV、TSV、JSON、JSONL、SQL INSERT、Excel互換HTML、Markdown を提供します。 大きい結果はディスク offload と windowed paging で保持できます。

BI / Sidebar

結果からすぐ chart を作る

BI は結果グリッドの Chart view と同じモデルを使いながら、active result の rows、columns、elapsed、sample 状態を sidebar 上で確認できます。 列は dimension / measure / time / field の role として整理されるため、 数値列や日付列を含む tabular result から bar / line / scatter を切り替え、 X、metric、aggregation、sort、limit をその場で調整できます。

Movable workbench views

Tables、Completion、History、Lakehouse、BI、Git は sidebar view として切り替えられます。 BI も他の view と同じように sidebar header の移動ボタンで左/右に寄せられるため、 VS Code の view container に近い作業配置を作れます。

Git

workbench 内の Git graph

Git drawer は commit graph、changes、branch 操作を一つのパネルにまとめます。 graph は subject/hash/author/ref 検索、branch / remote / tag filter、 commit detail、provider badge、repository color を表示します。

Keyboard navigation

graph list に focus すると ArrowUp / ArrowDown / Home / End で選択 commit を移動できます。

Security

秘密情報と接続経路

パスワードは session-only です。secure-store、redaction、audit log、 SSH / proxy / multi-hop は段階的に接続します。

Extensions

拡張 SDK

irodori.extension.json、TypeScript SDK、typed API、 driver、theme、result view を段階的に公開します。
