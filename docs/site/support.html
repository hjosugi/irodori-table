Bug reports

報告に含める情報

Irodori Table のバージョン、OS、データベース種別、再現手順、 redacted error、スクリーンショットを含めてください。

公開しない情報

パスワード、token、private key、database dump、顧客データ、 production connection URL は公開 Issue に含めないでください。

Scope

サポート範囲

インストール、起動、接続設定、SQL エディタ、Vim mode、結果グリッド、 schema metadata、import/export、extension SDK、release artifact を扱います。 本番データ復旧や非公開インフラの調査は対象外です。

Security

脆弱性報告

脆弱性の疑いがある場合、公開 Issue に詳細を書かず、 SECURITY.md の private disclosure 手順に従ってください。
