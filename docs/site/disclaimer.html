Preview

Development preview

現在は pre-1.0 です。機能、file format、extension API、supported database behavior は変更される可能性があります。

Database operations

データベース操作

database client は破壊的な SQL を実行できます。generated SQL、edit preview、 migration plan、import/export setting、target connection を確認し、 backup、transaction、read-only account、staging database、least privilege を使ってください。

AI-assisted features

AI 出力

AI-assisted SQL generation や explanation は不完全、不正確、非効率、または危険な出力を返す可能性があります。 生成結果は draft として扱い、実行前に構文、権限、query plan、業務影響を確認してください。

Third-party systems

第三者システム

database engine、cloud API、SSH/proxy transport、OS keychain、package manager、 optional model provider はそれぞれ独立して運営されています。可用性、互換性、価格、 データ処理、アクセス制御は各第三者の条件に従います。
