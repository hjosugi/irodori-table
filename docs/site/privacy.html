Summary

概要

アプリは Irodori の hosted account を必要としません。ユーザーが設定した database server、local file、proxy/SSH endpoint、任意の AI provider に接続します。

Local data

アプリがローカルで扱う情報

connection profile、SQL、query parameter、query result、metadata、 query plan、import/export file、UI preference、theme、keybinding、 query history、audit entry、diagnostic report を扱う場合があります。

Secrets

認証情報

通常の connection password は session-only の UI 入力として扱います。 保存される transport/proxy secret は、対応環境では OS keychain-backed secret store を使います。portable connection export は password field を redaction します。

Network

ネットワーク接続

接続先 database、SSH/proxy、cloud database API、local Ollama、 OpenAI-compatible endpoint、release/package update など、 ユーザーの設定や操作に応じて外部と通信します。

Third-party services

database provider、AI provider、proxy、OS keychain、package manager は それぞれの規約とプライバシーポリシーに従ってデータを処理します。

Diagnostics

ログと診断情報

privacy mode、redaction helper、redacted audit export は偶発的な漏えいを減らすための機能です。 診断情報やスクリーンショットを共有する前に、ユーザー自身で内容を確認してください。
