# Markdown Result Export

This example is dual-licensed as `MIT OR 0BSD`.

It contributes one command, one keybinding, and one result-grid action. The
command reads the active result grid and copies a Markdown table to the host
clipboard.

Run the local dev host from the repository root:

```sh
node packages/extension-sdk/bin/irodori-extension-dev.mjs examples/extensions/markdown-result-export --once
```
