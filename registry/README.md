# Registry

Generated or app-consumed reference data for Irodori Table.

Public docs live here:

<https://hjosugi.github.io/irodori-docs/>

## Files

- `data-source-support-status.md`: engine support snapshot.
- `cheatsheets/`: generated engine notes.
- `catalog/`: extension catalog data.
- `agent-workstreams.json`: workstream ownership for agent coordination.

Do not add long-form docs here. Put user-facing docs in `irodori-docs`.
